# 短信模板配置

<DifficultyBadge level="medium" />
<TimeEstimate :minutes="10" />

## 我需要这个吗？

**选填。** 前提是你已经在 [Phase 2](../phase2-notification/sms) 中开通了短信接口。

如果你需要发送短信验证码，就需要在短信平台配置模板内容。

---

## 操作步骤

### 第一步：登录短信平台

登录你注册的短信平台（如 [互亿无线](https://www.ihuyi.com)）。

### 第二步：创建短信模板

1. 进入 **「短信产品」** → **「模板管理」**
2. 点击 **「添加模板」**
3. 填写模板内容

**验证码模板示例：**

```
您的验证码为：{code}，{minute}分钟内有效，请勿泄露于他人。
```

:::warning 注意
- 模板中的变量格式要和平台要求一致（如 `{code}` 或 `##code##`）
- 模板内容需要符合平台规范，避免使用营销敏感词
- 审核通常需要 1~2 个工作日
:::

### 第三步：等待审核通过

提交模板后等待平台审核。审核通过后会分配一个**模板ID**，记录下来。

## 常见问题

:::details 模板审核不通过怎么办？
常见原因：
- 模板内容包含营销敏感词
- 变量格式不正确
- 内容描述不够清晰

建议参考平台的模板示例进行修改。
:::

:::details 可以配置多个模板吗？
可以，如果需要不同场景的短信（如注册验证码、订单通知等），可以分别创建模板。
:::
