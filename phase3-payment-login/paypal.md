# PayPal商户接入

<DifficultyBadge level="hard" />
<TimeEstimate :minutes="30" />

## 我需要这个吗？

**必填。** PayPal 是系统核心的在线支付方式，用户通过PayPal完成订单支付。

---

## 操作步骤

### 第一步：注册PayPal商家账号

1. 访问 [PayPal官网](https://www.paypal.com)
2. 点击 **「注册」**
3. 选择 **「商家账号」**（Business Account）
4. 填写邮箱和密码，完成注册

:::warning 注意
- 请使用稳定网络环境注册和登录
- 注册后请**不要频繁更换登录IP**，否则可能触发风控导致账号受限
:::

### 第二步：完成商家认证

1. 登录PayPal后，进入 **「账户设置」**
2. 按提示完成以下认证：
   - 邮箱验证
   - 手机验证
   - 商家信息填写（公司名称、地址等）
   - 银行账户或信用卡绑定

### 第三步：获取商户ID

1. 登录 [PayPal商家后台](https://www.paypal.com/businessmanage)
2. 点击页面右上角的**个人资料名**
3. 选择 **「账户设置」** → **「商家信息」**
4. 找到 **「商家ID」**（Merchant ID），这是一串由字母和数字组成的编号

:::tip 提示
商家ID通常显示为类似 `ABCDEF1234567` 的格式。
:::

### 第四步：（可选）配置API签名

如果你的集成方式需要API凭证，还需要获取以下信息：

1. 进入 **「账户设置」** → **「API访问」**
2. 点击 **「更新」** → 选择 **「请求API凭证」**
3. 记录以下信息：
   - API用户名
   - API密码
   - 签名

## 常见问题

:::details PayPal注册需要什么资料？
- 一个有效的邮箱地址
- 营业执照（企业账号）或身份证（个人账号）
- 银行账户信息用于提现
:::

:::details 商家账号和个人账号有什么区别？
商家账号支持接收信用卡付款和网站集成支付，个人账号功能有限。做电商必须使用商家账号。
:::

:::details 为什么账号被限制了？
PayPal风控较严格，常见触发原因包括：
- 注册后频繁更换登录IP
- 短时间内大额交易
- 信息填写不完整

建议使用稳定的网络环境，并尽快完成所有认证步骤。
:::

:::details 可以用国内PayPal吗？
可以，PayPal支持中国商家注册。提现时可以将资金转入国内银行账户。
:::
