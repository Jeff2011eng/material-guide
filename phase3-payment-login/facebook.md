# Facebook登录接入

<DifficultyBadge level="hard" />
<TimeEstimate :minutes="30" />

## 我需要这个吗？

**必填。** 用户可以通过Facebook账号直接登录你的网站，无需单独注册，提升用户体验。

---

## 操作步骤

### 第一步：登录Facebook开发者平台

1. 访问 [Facebook开发者平台](https://developers.facebook.com)
2. 使用你的Facebook账号登录
3. 如果是第一次使用，需要注册为开发者

:::warning 注意
- 请使用稳定网络环境操作
- 注册此应用后，请**不要频繁更换登录IP**，避免触发Facebook风控
- Facebook对开发者账号管理严格，违规操作可能导致账号被封
:::

### 第二步：创建应用

1. 点击 **「我的应用」** → **「创建应用」**
2. 选择应用类型：**「消费者」**（Consumer）
3. 填写应用信息：
   - **应用名称**：填写你的网站名称
   - **应用联系邮箱**：填写你的联系邮箱
   - **业务账户**：可选择跳过
4. 点击 **「创建应用」**

### 第三步：配置Facebook登录

1. 在应用面板中，找到 **「添加产品」**
2. 点击 **「Facebook登录」** 下的 **「设置」**
3. 选择平台：**「网站」**（Web）
4. 填写网站URL（技术支持会提供给你）
5. 保存设置

### 第四步：获取App ID和App Secret

1. 在应用面板的 **「设置」** → **「基本」** 页面
2. 你可以看到：
   - **App ID**：应用编号，直接显示
   - **App Secret**：点击 **「显示」** 按钮，输入密码验证后显示

:::warning 重要
App Secret 是重要凭证，请妥善保存，不要泄露给他人。
:::

### 第五步：提交应用审核

1. 在 **「应用审核」** 页面
2. 提交 `public_profile` 和 `email` 权限审核
3. 填写审核说明和使用场景

:::tip 提示
在审核通过前，只有开发者账号和测试账号可以使用Facebook登录。审核通常需要1~3个工作日。
:::

## 常见问题

:::details 我没有Facebook账号怎么办？
需要先注册一个Facebook个人账号，然后用这个账号去创建开发者应用。访问 [facebook.com](https://www.facebook.com) 注册。
:::

:::details 为什么创建应用后登录不生效？
应用需要通过Facebook审核才能让所有用户使用。在审核通过前，可以先将需要测试的Facebook账号添加为测试用户。
:::

:::details App ID和App Secret在哪里找？
登录开发者平台 → 你的应用 → 设置 → 基本。App ID直接显示，App Secret需要点击"显示"按钮。
:::

:::details 账号被封了怎么办？
Facebook封号后通常可以通过申诉恢复。建议：
- 使用稳定的网络环境
- 不要频繁更换登录设备
- 保持账号活跃，不要注册后就不管了
:::
