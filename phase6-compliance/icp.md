# ICP 备案

<DifficultyBadge level="hard" />
<TimeEstimate :minutes="60" />

## 我需要这个吗？

**视情况而定。**

- 服务器在**中国大陆**：**必须备案**，否则网站无法访问
- 服务器在**海外/港澳台**：**不需要备案**，可直接访问

:::warning 重要
ICP 备案审核周期较长（10~20 个工作日），如果需要备案请尽早开始。备案期间网站无法通过域名访问。
:::

---

## 操作步骤

### 第一步：确认是否需要备案

| 部署方式 | 是否需要备案 |
|----------|-------------|
| 服务器在中国大陆（阿里云、腾讯云等） | **需要** |
| 服务器在香港 | 不需要 |
| 服务器在海外（新加坡、美国等） | 不需要 |
| 使用 Cloudflare 等 CDN | 取决于源服务器位置 |

:::tip 建议
如果不确定服务器位置，联系技术支持确认。如果使用 Vercel 等海外平台部署，通常不需要备案。
:::

### 第二步：通过服务器提供商提交备案

根据你的服务器提供商选择对应备案入口：

| 提供商 | 备案入口 |
|--------|---------|
| 阿里云 | [备案系统](https://beian.aliyun.com) |
| 腾讯云 | [备案系统](https://cloud.tencent.com/product/ba) |
| 华为云 | [备案系统](https://beian.huaweicloud.com) |

### 第三步：准备备案材料

**个人备案：**
- 身份证正反面照片
- 域名证书
- 网站负责人手持身份证照片
- 核验单（服务商会提供模板）

**企业备案：**
- 营业执照照片
- 法人身份证正反面
- 网站负责人身份证正反面
- 网站负责人手持身份证照片
- 域名证书
- 核验单（加盖公章）

### 第四步：提交审核

1. 按服务商指引填写备案信息
2. 上传材料
3. 服务商初审（1~3 个工作日）
4. 提交至管局审核（5~20 个工作日）
5. 审核通过后获得 **ICP 备案号**

### 第五步：备案完成后

- 将 **ICP 备案号** 提供给技术支持
- 备案号需展示在网站底部
- 如有公安备案要求，还需在 [公安备案系统](https://beian.mps.gov.cn) 提交

## 常见问题

:::details 备案需要多久？
- 服务商初审：1~3 个工作日
- 管局审核：5~20 个工作日（各地不同）
- 总计约 10~25 个工作日
:::

:::details 备案期间网站能访问吗？
首次备案期间域名不能访问。如果是已有备案号新增网站，通常不影响已有网站。
:::

:::details 个人可以备案商业网站吗？
理论上不行。个人备案只能用于个人网站（博客、简历等）。商业用途（电商、代购等）需要企业备案。
:::
