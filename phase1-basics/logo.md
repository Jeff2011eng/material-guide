# 网站LOGO

<DifficultyBadge level="easy" />
<TimeEstimate :minutes="3" />

## 我需要这个吗？

**必填。** LOGO 是你网站的品牌标识，会显示在页头、登录页、浏览器书签等多个位置。

---

## 设计要求

| 要求 | 说明 |
|------|------|
| 格式 | PNG（优先）、JPG、SVG |
| 底色 | **透明底色**优先 |
| 形状 | **圆形**设计更佳（部分展示位为圆形裁切） |
| 尺寸 | 请提供**源文件/高清大图**，我们会自动裁剪适配 |

:::warning 注意
请勿提供截图或低分辨率图片，模糊的LOGO会严重影响网站专业形象。
:::

---

## 我要提供什么？

将LOGO源文件发送给技术支持即可：

---

## 常见问题

:::details 没有LOGO怎么办？
如果没有设计好的LOGO，可以暂时提供品牌名称的文字版本，我们先用文字占位，后续替换即可。
:::

:::details 支持什么格式？
PNG、JPG、SVG 均可，其中 PNG 透明底色效果最好。如果有 AI/PSD 源文件也可以一并提供。
:::

:::details 之后可以改吗？
可以，LOGO 支持在后台随时替换。
:::
