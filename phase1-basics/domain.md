# 域名

<DifficultyBadge level="easy" />
<TimeEstimate :minutes="10" />

## 我需要这个吗？

**必填。** 域名是网站的访问地址（如 `www.yourbrand.com`），是部署的前提条件。

---

## 操作步骤

### 第一步：选择域名

选择一个简短、好记、与品牌相关的域名。

| 建议后缀 | 适用场景 |
|----------|----------|
| `.com` | 国际通用，首选 |
| `.cn` | 面向国内市场 |
| `.net` | `.com` 被占用的备选 |
| `.co` | 简短好记的替代方案 |

:::tip 命名建议
- 优先使用品牌名称或缩写
- 尽量简短，避免拼写复杂
- 不要使用连字符（-）和数字，容易记错
:::

### 第二步：注册域名

选择以下任一域名注册商：

**国内注册商（推荐，实名认证方便）：**
- [阿里云（万网）](https://wanwang.aliyun.com) — 国内最大的域名注册商
- [腾讯云（DNSPod）](https://dnspod.cloud.tencent.com）

**国际注册商：**
- [Namecheap](https://www.namecheap.com)
- [GoDaddy](https://www.godaddy.com)
- [Cloudflare](https://www.cloudflare.com/products/registrar/)

:::warning 注意
国内注册商注册的域名需要完成**实名认证**（个人需身份证，企业需营业执照），通常 1~3 个工作日审核。
:::

### 第三步：完成实名认证（国内注册商）

如果在国内注册商购买，需要提交实名认证材料：
- **个人域名**：身份证正反面照片
- **企业域名**：营业执照照片

### 第四步：提供域名信息

将域名信息发送给技术支持，我们会完成 DNS 解析配置。

## 常见问题

:::details 域名大概多少钱？
`.com` 域名一般 50~70 元/年，`.cn` 域名 20~30 元/年。不同注册商价格略有差异。
:::

:::details 我不会配置 DNS 怎么办？
提供域名注册商的账号信息，技术支持可以帮你完成 DNS 解析配置。或者你也可以将域名管理权限临时授权给技术人员。
:::

:::details 已经有域名了还需要重新注册吗？
不需要，直接提供现有域名信息即可。技术支持会帮你调整 DNS 解析指向新的服务器。
:::

:::details .com 和 .cn 怎么选？
如果主要面向海外用户选 `.com`，主要面向国内用户可以两个都注册（`.com` 优先）。建议至少注册 `.com`。
:::
