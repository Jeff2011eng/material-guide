# 社交分享预览图（OG Image）

<DifficultyBadge level="easy" />
<TimeEstimate :minutes="5" />

## 我需要这个吗？

**选填。** 当用户把你的网站链接分享到微信、Facebook、WhatsApp、Telegram 等社交平台时，会显示一张预览图。没有这张图，分享时会显示空白或默认图片，影响专业感。

---

## 设计规范

| 规范项 | 要求 |
|--------|------|
| 建议尺寸 | **1200 × 630 px** |
| 格式 | PNG 或 JPG |
| 内容 | 品牌名 + 简短 slogan + LOGO |
| 分辨率 | 高清 |

:::tip 设计建议
- 背景使用品牌主色调
- 居中放置 LOGO 和网站名称
- 可以加一句简短的标语（如「全球好货，一站代购」）
- 文字要大，因为缩略图显示时很小
:::
