# 推广联盟背景图

<DifficultyBadge level="easy" />
<TimeEstimate :minutes="5" />

## 我需要这个吗？

**必填。** 推广联盟页面的顶部背景展示图，直接影响页面的视觉效果。

---

## 设计规范

| 规范项 | 要求 |
|--------|------|
| 建议尺寸 | **1920 × 500 px** |
| 格式 | PNG、JPG |
| 内容 | 与网站品牌/业务相关的背景图片 |
| 分辨率 | 高清，避免模糊 |

:::tip 提示
- 不同前台模板的尺寸可能有差异，如果不确定可以联系技术支持确认
- 建议提供**源文件**，方便后续裁剪适配
:::

## 常见问题

:::details 没有设计师怎么办？
可以提供一张你认为合适的高清图片（如风景、商品图等），我们协助裁剪适配。
:::

:::details 尺寸不对怎么办？
尽量按建议尺寸提供，如果无法精确匹配，提供尽可能大的高清图，我们会裁剪适配。
:::
