# 谷歌翻译API

<DifficultyBadge level="medium" />

## 我需要这个吗？

**选填。** 如果你需要提升多语言商品搜索的准确性，建议配置。

:::warning 注意
谷歌翻译API为付费服务，费用需自行承担。按使用量计费，一般用量下费用较低。
:::

## 操作步骤

### 第一步：注册Google Cloud账号

1. 访问 [Google Cloud Console](https://console.cloud.google.com)
2. 使用Google账号登录
3. 如果是第一次使用，需要创建一个项目

### 第二步：启用翻译API

1. 在控制台中搜索 **「Cloud Translation API」**
2. 点击 **「启用」**

### 第三步：创建API密钥

1. 进入 **「API和服务」** → **「凭据」**
2. 点击 **「创建凭据」** → **「API密钥」**
3. 记录生成的API密钥

:::tip 提示
建议为API密钥设置HTTP引用来源限制，只允许你的网站使用，防止密钥被滥用。
:::

### 第四步：配置计费

1. 进入 **「计费」** 页面
2. 绑定信用卡（Google Cloud需要绑定付款方式）
3. 翻译API有免费额度，超出后按量计费

## 常见问题

:::details 费用大概是多少？
谷歌翻译API提供每月前50万个字符免费，超出后约 $20/百万字符。普通电商网站搜索翻译的用量通常不会超过免费额度。
:::

:::details 需要科学上网吗？
注册和配置时需要，但配置完成后用户使用翻译功能不需要。
:::
