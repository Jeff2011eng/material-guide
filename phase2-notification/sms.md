# 短信接口

<DifficultyBadge level="medium" />
<TimeEstimate :minutes="15" />

## 我需要这个吗？

**选填。** 只有在以下场景才需要配置：
- 你希望用户注册/登录时使用**手机短信验证码**
- 你希望通过短信向用户发送订单通知

如果你不需要短信验证功能，可以跳过此项。

:::warning 注意
短信服务为第三方付费服务，费用需要自行承担。
:::

---

## 操作步骤

### 注册互亿无线短信平台（推荐平台）

**第一步：访问平台**

访问 [互亿无线官网](https://www.ihuyi.com)，点击「免费注册」。

**第二步：注册账号**

1. 填写手机号码，获取验证码
2. 设置登录密码
3. 完成注册

**第三步：实名认证**

登录后根据提示完成实名认证（个人或企业均可），认证通过后才能使用短信服务。

**第四步：充值**

根据业务量选择充值金额。新用户通常有免费测试额度，可以先测试再充值。

**第五步：获取API信息**

1. 进入 **「API接口」** 或 **「开发者配置」** 页面
2. 记录以下信息：
   - **API ID**（账号标识）
   - **API KEY**（密钥）

:::tip 提示
平台同时提供 HTTP 和 WebService 接口，我们系统使用的是 HTTP 接口。
:::

## 常见问题

:::details 短信费用大概是多少？
互亿无线的验证码短信大约 0.04~0.05 元/条，具体价格以平台实际报价为准。充值金额根据你的用户量决定，一般 100 元可以用很久。
:::

:::details 可以用其他短信平台吗？
可以，只要平台支持标准的 HTTP API 接口即可。但推荐使用互亿无线，因为我们的系统已经做了适配。
:::

:::details 配置后可以改吗？
可以，短信接口配置支持在后台自行修改。
:::
