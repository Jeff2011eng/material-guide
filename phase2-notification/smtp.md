# SMTP邮箱配置

<DifficultyBadge level="medium" />
<TimeEstimate :minutes="15" />

## 我需要这个吗？

**必填。** 系统需要通过邮箱发送以下通知：
- 会员注册确认邮件
- 订单采购到库通知
- 密码找回邮件
- 其他系统通知

---

## 什么是SMTP？

SMTP（Simple Mail Transfer Protocol）是发送邮件的标准协议。你只需要一个支持SMTP的邮箱账号，系统就能通过这个邮箱自动发送通知邮件。

---

## 操作步骤

### 方案一：使用QQ邮箱（推荐，国内最方便）

:::details 点击展开QQ邮箱配置教程

**第一步：登录QQ邮箱**

访问 [mail.qq.com](https://mail.qq.com)，使用你的QQ账号登录。

**第二步：开启SMTP服务**

1. 点击页面上方的 **「设置」** → **「账户」**
2. 向下滚动找到 **「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务」**
3. 找到 **「SMTP服务」**，点击 **「开启」**
4. 按提示用手机发送短信进行验证
5. 验证成功后，页面会显示一个 **「授权码」**（16位字母）

:::warning 重要
请务必保存好这个授权码，它就是你的SMTP密码。授权码只显示一次，如果忘记了需要重新生成。
:::

**第三步：记录配置信息**

| 配置项 | 值 |
|--------|-----|
| SMTP服务器 | `smtp.qq.com` |
| SMTP端口 | `465`（SSL加密） |
| SMTP用户名 | 你的QQ邮箱地址（如 `123456789@qq.com`） |
| SMTP密码 | 上一步获取的**授权码**（不是QQ密码） |

:::

### 方案二：使用163邮箱

:::details 点击展开163邮箱配置教程

**第一步：登录163邮箱**

访问 [mail.163.com](https://mail.163.com)，使用你的网易账号登录。

**第二步：开启SMTP服务**

1. 点击页面上方的 **「设置」** → **「POP3/SMTP/IMAP」**
2. 找到 **「SMTP服务」**，点击 **「开启」**
3. 按提示发送短信进行验证
4. 验证成功后，页面会显示**「授权密码」**

**第三步：记录配置信息**

| 配置项 | 值 |
|--------|-----|
| SMTP服务器 | `smtp.163.com` |
| SMTP端口 | `465`（SSL加密） |
| SMTP用户名 | 你的163邮箱地址（如 `yourname@163.com`） |
| SMTP密码 | 上一步获取的**授权密码** |

:::

### 方案三：使用Gmail

:::details 点击展开Gmail配置教程

**第一步：登录Google账号**

访问 [gmail.com](https://gmail.com)，登录你的Google账号。

**第二步：开启两步验证**

1. 访问 [Google账号安全设置](https://myaccount.google.com/security)
2. 确保 **「两步验证」** 已开启（这是创建应用密码的前提）

**第三步：创建应用密码**

1. 访问 [应用密码页面](https://myaccount.google.com/apppasswords)
2. 选择「邮件」，输入一个自定义名称（如「代购网站」）
3. 点击「创建」，会生成一个16位密码

**第四步：记录配置信息**

| 配置项 | 值 |
|--------|-----|
| SMTP服务器 | `smtp.gmail.com` |
| SMTP端口 | `465`（SSL加密） |
| SMTP用户名 | 你的Gmail地址（如 `yourname@gmail.com`） |
| SMTP密码 | 上一步生成的**应用密码** |

:::

## 常见问题

:::details 用哪个邮箱比较好？
- 国内用户推荐 **QQ邮箱**，配置最简单
- 如果是企业客户，推荐使用**企业邮箱**，更正式
- Gmail 需要科学上网，不推荐国内客户使用
:::

:::details 为什么我的SMTP密码不是邮箱登录密码？
出于安全考虑，大部分邮箱平台要求生成**专用授权码/应用密码**用于第三方发送邮件。这个授权码和你的邮箱登录密码是独立的。
:::

:::details 发送的邮件会被识别为垃圾邮件吗？
正常情况下不会。系统发送的都是注册确认、订单通知等用户主动触发的邮件。建议使用企业邮箱以获得更好的送达率。
:::
